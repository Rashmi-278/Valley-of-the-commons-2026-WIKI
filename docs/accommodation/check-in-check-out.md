# Check-in / Check-out

---

## Check-in Procedure

### Full Residents (24 August)

| Step | Detail |
|------|--------|
| **1. Arrival window** | Define check-in window (e.g., 14:00-20:00 on 24 Aug) |
| **2. Welcome desk** | Set up in main Hall - staffed by team member |
| **3. Verify identity** | Cross-reference with participant database |
| **4. Confirm payment** | Must be fully paid before key handoff |
| **5. Room assignment** | Hand over room key/code + room info sheet |
| **6. Welcome packet** | Physical folder with schedule, map, Wi-Fi info, kitchen rules, emergency contacts |
| **7. Orientation briefing** | Short tour of facilities |
| **8. Sign house rules** | Participant signs acknowledgment of code of conduct |
| **9. Point of contact** | Collect participant's phone number / Telegram handle |
| **10. First meeting** | 5 minutes with team and whole group |

### Weekly Visitors

- Same process, but at the start of their booked week
- Lighter orientation (can be 1-on-1)
- Assign a "buddy" from full residents if possible

---

## Check-out Procedure

### Full Residents (20 September)

| Step | Detail |
|------|--------|
| **1. Checkout window** | Define window (e.g., by 12:00 on 21 Sep) |
| **2. Room inspection** | Quick check of room condition |
| **3. Key return** | Collect all keys/access cards |
| **4. Feedback form** | Hand out or email link to post-event survey |
| **5. Settle outstanding** | Any unpaid balances |
| **6. Farewell** | Closing ceremony on evening of 20 Sep |

### Weekly Visitors

- Checkout by morning of the last day of their booked week
- Same room inspection + key return process

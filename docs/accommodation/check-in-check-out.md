# Check-in / Check-out

- - -

## Check-in Procedure

### Full Residents (23 August)

| Step                        | Detail                                                                            |
| --------------------------- | --------------------------------------------------------------------------------- |
| **1. Arrival window**       | 09:00-20:00 Aug 23                                                                |
| **2. Welcome desk**         | Set up in lounge - staffed by team member                                         |
| **3. Verify identity**      | Cross-reference with participant database                                         |
| **4. Confirm payment**      | Must be fully paid before check in                                                |
| **5. Welcome packet**       | Physical folder with schedule, map, Wi-Fi info, kitchen rules, emergency contacts |
| **6. Orientation briefing** | Short tour of facilities                                                          |
| **7. Sign house rules**     | Participant signs acknowledgment of code of conduct                               |
| **8. Point of contact**     | Collect participant's phone number / Telegram handle                              |
| **9. First meeting**        | 5 minutes with team and whole group on eve of Aug 23                              |

### Weekly Visitors

* Same process, but at the start of their booked week
* Lighter orientation (can be 1-on-1)
* Assign a "buddy" from full residents if possible

- - -

## Check-out Procedure

### Full Residents (20 September)

| Step                      | Detail                                          |
| ------------------------- | ----------------------------------------------- |
| **1. Checkout window**    | Define window (e.g., by 12:00 on 20 Sep)        |
| **2. Key return**         | Collect all keys/access cards                   |
| **3. Feedback form**      | Hand out or email link to post-event survey     |
| **4. Settle outstanding** | Any unpaid balances                             |
| **5. Farewell**           | Closing ceremony in the morning/ noon of 20 Sep |

### Weekly Visitors

* Checkout by morning of the last day of their booked week

* Valley map + Hub floor plan
* Wi-Fi credentials card
* Emergency contact card
